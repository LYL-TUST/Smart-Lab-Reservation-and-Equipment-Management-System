# 智能实验室预约与设备管理系统 - 完整实施计划

## 项目概述

实现一个完整的实验室预约与设备管理系统，支持管理员、教师、学生三类用户，实现单次预约、多次预约、课程绑定预约等功能，包含智能冲突检测算法。

## 技术栈

- **后端**: Spring Boot 2.7+ + MyBatis-Plus + MySQL + Spring Security + JWT
- **前端**: Vue 3 + Vite + Element Plus + Pinia + Vue Router
- **测试**: JUnit 5 + Mockito
